# El-Yunque-Biodiversity-App
Unfinished Adobe XD application that organizes the wildlife and plant species that live throughout the rainforest. It presents them clearly through pictures; thus, the user can easily search for a species that interests them while roaming the rainforest. Once chosen through a click, it presents facts and references for more information about the species. Uses Fun Facts and Trivia to present data in a more enjoyable way.
